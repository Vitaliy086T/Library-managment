## Задача: "Библиотека книг"

Цель: Создать простую систему управления библиотекой книг, используя принципы объектно-ориентированного программирования (ООП).

Описание:

1. Книга:
* Имеет следующие атрибуты:
* Название (строка)
* Автор (строка)
* Год издания (целое число)
* ISBN (строка)
* Доступность (булевое значение: true - доступна, false - недоступна)
* Методы:
* `getНазвание()`: возвращает название книги.
* `getАвтор()`: возвращает автора книги.
* `getГодИздания()`: возвращает год издания книги.
* `getISBN()`: возвращает ISBN книги.
* `isДоступна()`: возвращает доступность книги.
* `setДоступность(bool доступность)`: устанавливает доступность книги.

2. Библиотека:
* Имеет следующие атрибуты:
* Список книг (вектор объектов класса "Книга")
* Методы:
* `добавитьКнигу(Книга книга)`: добавляет новую книгу в список.
* `удалитьКнигу(string ISBN)`: удаляет книгу из списка по ISBN.
* `найтиКнигу(string название)`: возвращает указатель на книгу с заданным названием, или null, если книги нет.
* `вывестиСписокКниг()`: выводит информацию о всех книгах в списке.

Дополнительные задачи:

* Добавьте возможность сортировки книг по названию, автору, году издания.
* Добавьте возможность поиска книг по автору.
* Добавьте возможность добавления пользователей и их заимствований.
* Реализуйте возможность выдачи книг пользователям и их возврата.

Рекомендации:

* Используйте классы для представления книг и библиотеки.
* Используйте методы для реализации операций с книгами и библиотекой.
* Применяйте принципы инкапсуляции, наследования и полиморфизма (если применимо).
